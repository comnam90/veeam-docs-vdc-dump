# Licensing


Veeam Data Cloud for AWS is licensed on a Back-End Terabyte (BETB) basis. Since Veeam Data Cloud for AWS compresses all backed-up data when saving it to backup repositories, subscription costs are based on the actual size of the data after compression.

|  |
| --- |
| Note |
| The compression rate depends on the type and structure of source data and usually varies from 50% to 60%. |

You can monitor the amount of space consumed by backups stored in backup repositories on the AWS page; for more information, see [Viewing Dashboards](aws_dashboard.md#home_page_dashboard).

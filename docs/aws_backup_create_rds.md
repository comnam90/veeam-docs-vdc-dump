# Creating RDS Backup Policies


To create a backup policy, complete the following steps:

1. [Launch the Add RDS Policy wizard](aws_backup_create_rds_launch.md).
2. [Specify a backup policy name and description](aws_backup_create_rds_name.md).
3. [Configure backup source settings](aws_backup_create_rds_source_settings.md).
4. [Configure protection settings](aws_backup_create_rds_sla.md).
5. [Specify processing settings](aws_backup_create_rds_processing_settings.md).
6. [Review default policy settings](aws_backup_create_rds_default_settings.md).
7. [Enable AWS tags assigning](aws_backup_create_rds_tags.md).
8. [Specify notification settings](aws_backup_create_rds_notifications.md).
9. [Finish working with the wizard](aws_backup_create_rds_finish.md).

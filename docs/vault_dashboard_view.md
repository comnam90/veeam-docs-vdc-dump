# Accessing Veeam Data Cloud Vault


To access Veeam Data Cloud Vault, do either of the following:

* On the Veeam Data Cloud overview page, on the Veeam Data Cloud Vault widget, click Manage.
* Click Veeam Data Cloud Vault logo in the left menu.

|  |
| --- |
| Important |
| To access Veeam Data Cloud Vault, you must have an active subscription. To get a subscription, click Request a Subscription in the Veeam Data Cloud Vault widget. |

The Vault page displays the following widgets.

* The Subscriptions widget displays information on state of your Veeam Data Cloud Vault subscriptions. The widget contains the following elements:

* The Subscriptions counter shows the total amount of available subscriptions and number of unused subscriptions.
* The Tenants Onboarded counter shows the total amount of tenants onboarded in your subscriptions.

* The Subscription Edition Details widget displays information about the storage use.
* The tenant widget displays information on Veeam Data Cloud Vault tenants. To learn more about this widget, see [Viewing Tenants](vault_tenants_view.md).

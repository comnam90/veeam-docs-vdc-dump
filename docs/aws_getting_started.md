# Getting Started with Veeam Data Cloud for AWS


To start working with Veeam Data Cloud for AWS, perform the following steps:

1. [Obtain access to Veeam Data Cloud and sign in to the portal](accessing_vdc.md).
2. [[Optional] Invite additional users to your Veeam Data Cloud organization.](users_manage.md)
3. [Check limitations and prerequisites for Veeam Data Cloud for AWS](aws_planning_and_preparation.md).
4. [Create tenants to access AWS services and resources](aws_tenants.md).
5. [[Optional] Add database accounts to access databases of PostgreSQL DB instances](aws_settings_database_accounts.md).
6. [Create backup policies to protect your AWS resources](aws_backup.md).

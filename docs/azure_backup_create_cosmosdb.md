# Creating Cosmos DB Backup Policies


Use the New Cosmos DB Policy wizard to create a backup policy to protect Cosmos DB accounts.

|  |
| --- |
| Important |
| Before you create the first Cosmos DB backup policy, make sure that your service account has all necessary permissions. To do that, complete the [Edit account](azure_settings_accounts_service_edit.md) wizard, even if you have already done so for other backup policies types. |

1. [Launch the New Cosmos DB Policy wizard](azure_backup_create_cosmosdb_launch.md).
2. [Specify the policy details](azure_backup_create_cosmosdb_name.md).
3. [Specify the policy source](azure_backup_create_cosmosdb_source.md).
4. [Specify the policy protection settings](azure_backup_create_cosmosdb_sla.md).
5. [Complete the backup policy configuration](azure_backup_create_cosmosdb_complete.md).

# Step 1. Launch New Azure SQL Policy Wizard


To launch the New Azure SQL Policy wizard, do the following:

1. In the Backups section of the main menu, select Policies.
2. Select Databases > Azure SQL and click New Policy.

[![Launch New Azure SQL Policy Wizard](images/azure_backup_create_sql_launch.png)](images/azure_backup_create_sql_launch.png "Launch New Azure SQL Policy Wizard")

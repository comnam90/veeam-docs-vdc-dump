# Entra ID Group Restore


If you want to restore Microsoft Entra ID group data, you can do either of the following:

* [Restore entire groups from a backup or Entra ID recycle bin](entra_id_restore_groups.md). You can restore one or multiple groups.
* [Restore specific group properties](entra_id_restore_group_prop.md). You can restore one or multiple properties.

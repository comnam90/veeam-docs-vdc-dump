# Step 6. Review Default Policy Settings


At the Summary step of the wizard, do either of the following:

* To complete the backup policy configuration using the default settings, click Finish .

[![Review Default Policy Settings](images/azure_backup_create_sql_review_default.webp)](images/azure_backup_create_sql_review_default.webp "Review Default Policy Settings")

* To configure additional settings for the policy, turn on the Configure advanced settings toggle — in this case, the step name will change to Default Settings — and click Next.

[![Activate Advanced Policy Settings](images/azure_backup_create_sql_review_advanced.webp)](images/azure_backup_create_sql_review_advanced.webp "Activate Advanced Policy Settings")

# Step 1. Launch Add EFS Policy Wizard


To launch the Add EFS Policy wizard, do the following:

1. On the AWS page, locate a tenant that has access to resources that you want to back up, and click Manage in the Actions column.
2. On the tenant administration page, navigate to Policies > File Systems > EFS and click Add.

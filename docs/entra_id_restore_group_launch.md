# Step 1. Launch Restore Groups Wizard

In this article

To launch the Restore Groups wizard, do the following:

1. On the Entra ID page, click the name of the tenant you want to manage.
2. Select Objects.
3. Make sure that the Groups tab is selected.
4. In the list of groups, select those you want to restore.

|  |
| --- |
| Tip |
| Consider the following:   * To find a group by its display name, you can use the search field.  * To show more group properties, click the menu icon and select the properties you want to show. * To filter groups by group type or membership type, click Filter and select required values. * To export the list of groups for future references and imports, click Export to and select the format in which you want to save the list. Veeam Data Cloud will export all backed-up groups and their properties that can be shown in the restore window. |

1. To launch the restore wizard, click Restore.

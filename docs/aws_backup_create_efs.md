# Creating EFS Backup Policies


To create a backup policy, complete the following steps:

1. [Launch the Add EFS Policy wizard](aws_backup_create_efs_launch.md).
2. [Specify a backup policy name and description](aws_backup_create_efs_name.md).
3. [Configure backup source settings](aws_backup_create_efs_source_settings.md).
4. [Configure protection settings](aws_backup_create_efs_protection_settings.md).
5. [Review default policy settings](aws_backup_create_efs_default_settings.md).
6. [Enable AWS tags assigning](aws_backup_create_efs_tags.md).
7. [Specify notification settings](aws_backup_create_efs_notifications.md).
8. [Finish working with the wizard](aws_backup_create_efs_finish.md).

# Entra ID Administrative Units Restore


If you want to restore Microsoft Entra ID administrative unit data, you can do either of the following:

* [Restore entire administrative units from a backup or Entra ID recycle bin](entra_id_restore_unit.md). You can restore one or multiple administrative units.
* [Restore specific administrative unit properties](entra_id_restore_unit_prop.md). You can restore one or multiple properties.

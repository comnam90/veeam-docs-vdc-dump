# Licensing


Veeam Data Cloud Vault is licensed through a subscription model on a per-terabyte (TB) basis. Each Veeam Data Cloud Vault subscription includes the storage capacity, read and write API operations, and outbound data transfer. You can scale up the capacity at any time or obtain additional subscriptions to extend available options.

Veeam offers annual and multi-year subscriptions that include a fixed amount of storage capacity. Any storage use over that capacity is subject to monthly charges.

Veeam Data Cloud Vault is available in several editions that differ in durability, supported regions, read and restore allowances, and the cloud environment where Veeam Data Cloud Vault stores your data. Each subscription corresponds to one product edition. For details, see [Veeam Data Cloud Vault Editions](vault_editions.md).

For more information on obtaining a Veeam Data Cloud Vault subscription, see [Obtaining Veeam Data Cloud Vault](vault_obtain_product.md).

Subscription Expiration

If your subscription expires and you do not renew it, Veeam Data Cloud will retain your storage and data according to the Veeam Data Cloud service agreement. For more information, see [this Veeam article](https://www.veeam.com/legal/veeam-data-cloud-service-agreement.html).

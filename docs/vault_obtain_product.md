# Obtaining Veeam Data Cloud Vault


Veeam offers several scenarios for obtaining Veeam Data Cloud Vault. For more information on your options, see the [Veeam Data Cloud Vault product page](https://www.veeam.com/products/veeam-data-cloud/purchasing-options.html?ad=hero-banner#vault-tab).

To obtain Veeam Data Cloud Vault, consider the following scenarios:

* Microsoft Azure Marketplace

To use this scenario, go to the Microsoft Azure Marketplace portal from the Veeam Data Cloud Vault page on the Veeam website and follow the subscription purchase procedure.

* AWS Marketplace

To use this scenario, go to the AWS Marketplace portal from the Veeam Data Cloud Vault page on the Veeam website and follow the subscription purchase procedure.

* Veeam Online Store

To use this scenario, go to the [Veeam Online Store](https://store.veeam.com/), click Buy Now in the Veeam Vault section and follow the subscription purchase procedure.

* Veeam reseller

To use this scenario, contact a Veeam partner from the Veeam Data Cloud Vault page on the Veeam website.

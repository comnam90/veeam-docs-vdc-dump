# Adding Salesforce Tenants


To start protecting your Salesforce data, you must add a Salesforce tenant. Before you start adding the tenant, check [Considerations and Limitations](sf_limitations.md).

|  |
| --- |
| Note |
| * If your tenants use more than 120% of licensed users in your Veeam Data Cloud for Salesforce subscription, Veeam Data Cloud will display a warning that you have exceeded the limit. However, it will allow you to add a new tenant. * If your tenants use more than 200% of licensed users in your Veeam Data Cloud for Salesforce subscription, Veeam Data Cloud will prevent you from adding a new Salesforce tenant. |

To add a new Salesforce tenant, use the Add Salesforce Tenant wizard:

1. [Launch the Add Salesforce Tenant wizard](sf_tenants_add_launch.md).
2. [Connect Veeam Data Cloud to your Salesforce tenant](sf_tenants_add_connect.md).
3. [Specify Salesforce tenant settings](sf_tenants_add_region.md).
4. [Configure the default backup policy](sf_tenants_add_backup.md).
5. [Finish working with the wizard](sf_tenants_add_summary.md).

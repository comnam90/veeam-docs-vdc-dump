# Restoring Azure VMs


You can restore Azure VM data from an image snapshot or image-level backup by using the following restore options:

* [Restoring Entire VMs](azure_restore_vm_entire.md) — use this option to restore an entire Azure VM.
* [Restoring VM Disks](azure_restore_vm_disks.md) — use this option to restore virtual disks attached to an Azure VM.
* [Restoring VM Files](azure_restore_vm_files.md) — use this option to restore individual files and folders of Azure VM.

You can restore Azure VM data to the most recent state or to any available restore point.

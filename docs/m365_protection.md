# Microsoft 365 Protection


Veeam Data Cloud for Microsoft 365 is a backup as a service solution developed for data protection for Microsoft 365. With Veeam Data Cloud for Microsoft 365, you can back up and restore the data of the following Microsoft 365 services:

* Microsoft Exchange Online
* Microsoft SharePoint Online
* Microsoft OneDrive
* Microsoft Teams

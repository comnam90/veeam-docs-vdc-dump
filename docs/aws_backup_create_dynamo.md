# Creating DynamoDB Backup Policies


To create a backup policy, complete the following steps:

1. [Launch the Add DynamoDB Policy wizard](aws_backup_create_dynamo_launch.md).
2. [Specify a backup policy name and description](aws_backup_create_dynamo_name.md).
3. [Configure backup source settings](aws_backup_create_dynamo_source_settings.md).
4. [Configure protection settings](aws_backup_create_dynamo_protection_settings.md).
5. [Review default policy settings](aws_backup_create_dynamo_default_settings.md).
6. [Enable AWS tags assigning](aws_backup_create_dynamo_tags.md).
7. [Specify notification settings](aws_backup_create_dynamo_notifications.md).
8. [Finish working with the wizard](aws_backup_create_dynamo_finish.md).

# Requesting Subscriptions


You can request one or multiple Veeam Data Cloud subscriptions for your customer in VCSP Pulse.

To request new subscriptions use the New Subscription Request wizard.

1. [Launch the New Subscription Request wizard](sp_subscriptions_request_launch.md).
2. [Specify a customer for new subscriptions](sp_subscriptions_request_customer.md).
3. [Specify customer details](sp_subscriptions_request_details.md).
4. [Specify products for the subscription request](sp_subscriptions_request_products.md).

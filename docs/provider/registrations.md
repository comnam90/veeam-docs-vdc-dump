# Product Registrations


Product registration allows you to connect on-premises Veeam products, such as Veeam Backup & Replication, to Veeam Data Cloud. After you register a product, it can use Veeam Data Cloud services within the scope of availability that you specify.

You register and manage products on the Product Registration tab of the Settings page. For each registered product, the tab shows the product type, host name, status, organization and registration date.

This feature is available only for users with the OrganizationAdmin role assigned.

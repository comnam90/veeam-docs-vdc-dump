# Users


Veeam Data Cloud allows you to create, edit and delete users in your Veeam Data Cloud organization and Veeam Data Cloud organizations of your customers. You can assign multiple roles to a user and configure a role scope. User roles define the operations that the user can perform, including backup, restore and user management. For more information on Veeam Data Cloud roles, see [Roles](users_roles.md).

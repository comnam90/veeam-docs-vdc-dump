# Log Record Action IDs


This page contains a list of all Action IDs that can appear in a log record from a Veeam Data Cloud for Microsoft 365 syslog server.

| Name | Description |
| --- | --- |
| BulkRestoreOutlooks | Bulk restore Outlook mailboxes. |
| RestoreOutlook | Restore Outlook mailbox to its original location. |
| RestoreOutlookToAnother | Restore Outlook mailbox to another user mailbox. |
| RestoreOutlookFolder | Restore Outlook folder to its original location. |
| RestoreOutlookFolderToAnother | Restore Outlook folder to another user mailbox. |
| RestoreOutlookItems | Restore Outlook items to their original location. |
| RestoreOutlookItemsToAnother | Restore Outlook items to another user mailbox. |
| ViewEmailBody | View Outlook email. |
| RestoreOutlookToLocal | Download Outlook mailbox. |
| RestoreOutlookToAzureStorage | Download Outlook mailbox or Outlook folder in the background. |
| RestoreOutlookItemsToLocal | Download Outlook items. |
| RestoreOutlookItemsToAzureStorage | Download Outlook items in the background. |
| RestoreOneDrive | Restore OneDrive to its original location. |
| BulkRestoreOneDrives | Bulk restore OneDrive. |
| RestoreOneDriveFolders | Restore OneDrive folders to their original location. |
| RestoreOneDriveDocuments | Restore OneDrive documents to their original location. |
| RestoreOneDriveToAnother | Restore OneDrive to another user OneDrive. |
| RestoreOneDriveFoldersToAnother | Restore OneDrive folders to another user OneDrive. |
| RestoreOneDriveDocumentsToAnother | Restore OneDrive documents to another user OneDrive. |
| RestoreOneDriveItems | Restore OneDrive items to their original location. |
| RestoreOneDriveToLocal | Download OneDrive. |
| RestoreOneDriveToAzureStorage | Download OneDrive, OneDrive folder, OneDrive document in the background. |
| RestoreOneDriveDocumentsToLocal | Download OneDrive documents. |
| RestoreOneDriveFoldersToLocal | Download OneDrive folders. |
| RestoreOneDriveDocumentsToAzureStorage | Download multiple OneDrive documents in the background. |
| RestoreOneDriveFoldersToAzureStorage | Download multiple OneDrive folders in the background. |
| CreateRestoreSession | Create restore session. |
| RestoreSharePointSite | Restore SharePoint site to its original location. |
| RestoreSharePointSiteDocuments | Restore SharePoint site documents to their original location. |
| RestoreSharePointSiteDocumentsToAnother | Restore SharePoint site documents to another site. |
| RestoreSharePointSiteFolder | Restore SharePoint site folder to its original location. |
| RestoreSharePointSiteFolderToAnother | Restore SharePoint site folder to another site. |
| RestoreSharePointSiteFolders | Restore SharePoint site folders to their original location. |
| RestoreSharePointSiteFoldersToAnother | Restore SharePoint site folders to another site. |
| RestoreSharePointSiteItems | Restore SharePoint site items to their original location. |
| RestoreSharePointSiteItemsToAnother | Restore SharePoint site items to another site. |
| RestoreSharePointSiteLibrary | Restore SharePoint site library to its original location. |
| RestoreSharePointSiteLibraryToAnother | Restore SharePoint site library to another site. |
| RestoreSharePointSiteList | Restore SharePoint site list to its original location. |
| RestoreSharePointSiteListToAnother | Restore SharePoint site list to another site. |
| RestoreSharePointToLocal | Download SharePoint site. |
| BulkRestoreTeams | Bulk restore Teams. |
| RestoreTeams | Restore Teams to their original location. |
| RestoreTeamsChannels | Restore Teams channels to their original location. |
| RestoreTeamsPosts | Restore Teams posts to their original location. |
| RestoreTeamsFiles | Restore Teams files to their original location. |
| RestoreTeamsTabs | Restore Teams tabs to their original location. |
| RestoreTeamsToLocal | Download Teams. |
| RestoreTeamsToAzureStorage | Download Teams posts or Teams files in the background. |
| ResumeOrganization | Resume organization when its status is Canceling, Canceled or Deleting. |
| CancelSubscription | Cancel Veeam Data Cloud for Microsoft 365 subscription. |
| CreatePortalSession | Create Stripe customer portal session. For more information, see [this article](https://docs.stripe.com/api/customer_portal/sessions). |

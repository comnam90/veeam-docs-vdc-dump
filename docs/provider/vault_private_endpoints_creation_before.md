# Before You Begin


Before you configure a private endpoint connection, check the following prerequisites:

* You can only use private endpoints for Azure edition storage vaults.
* You must have an access to the Microsoft Azure portal.
* You must assign appropriate Azure role-based access control (RBAC) permissions to be able to create private endpoints. To learn more, see [Microsoft documentation](https://learn.microsoft.com/en-us/azure/private-link/rbac-permissions#private-endpoint).

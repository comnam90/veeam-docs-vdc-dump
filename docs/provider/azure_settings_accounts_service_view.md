# Viewing Azure Service Accounts


To view information about the service account that has access to Azure resources, select Azure Service Accounts.

[![Viewing Azure Service Accounts](images/azure_settings_accounts_service.webp)](images/azure_settings_accounts_service.webp)

Click View to see the list of enabled subscriptions.

[![Viewing Azure Service Accounts](images/azure_settings_accounts_service_view.webp)](images/azure_settings_accounts_service_view.webp)

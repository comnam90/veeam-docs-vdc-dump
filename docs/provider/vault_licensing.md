# Licensing


Veeam Data Cloud Vault is licensed through a subscription model on a per-terabyte (TB) basis. Each Veeam Data Cloud Vault subscription includes the storage capacity, read and write API operations, and outbound data transfer.

Veeam Data Cloud Vault is available in several editions that differ in durability, supported regions, read and restore allowances, and the cloud environment where Veeam Data Cloud Vault stores your data. Each subscription corresponds to one product edition. For details, see [Veeam Data Cloud Vault Editions](vault_editions.md).

VCSP Pulse reports Veeam Data Cloud Vault usage per subscription. To track per-customer usage in the VCSP Pulse monthly usage report, request a Veeam Data Cloud Vault subscription for each customer individually. For more information on requesting a subscription for your customers, see [Requesting Subscriptions](sp_subscriptions_request.md).

VCSP Pulse rounds storage consumption up to the nearest TB. For example, the consumption of 1.05 TB in Veeam Data Cloud Vault is rounded to 2 TB in the VCSP Pulse monthly usage report.

Subscription Expiration

If your subscription expires and you do not renew it, Veeam Data Cloud will retain your storage and data according to the Veeam Data Cloud service agreement. For more information, see [this Veeam article](https://www.veeam.com/legal/veeam-data-cloud-service-agreement.html).

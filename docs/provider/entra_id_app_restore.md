# Entra ID Application Restore


If you want to restore Microsoft Entra ID application or service principal data, you can do either of the following:

* [Restore entire applications and service principals from a backup or Entra ID recycle bin](entra_id_restore_app.md). You can restore one or multiple objects.
* [Restore specific application or service principal properties](entra_id_restore_app_prop.md). You can restore one or multiple properties.

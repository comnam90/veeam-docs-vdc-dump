# Adding Microsoft Entra ID Tenants


To start protecting the customer Microsoft Entra ID tenant data, add the Microsoft Entra ID tenant of your customer to Veeam Data Cloud. Before you start adding the tenant, check [Considerations and Limitations](entra_id_limitations.md).

To add a Microsoft Entra ID tenant, use the Add Entra ID tenant wizard:

1. [Launch the Add Entra ID tenant wizard](sp_entra_id_tenant_add_launch.md).
2. [Authorize access to your Entra ID domain](entra_id_tenant_add_auth.md).
3. [Specify where your backed up data will be stored](entra_id_tenant_add_region.md).
4. [Specify a retention period for your backups](entra_id_tenant_add_backup.md).
5. [Finish working with the Add Entra ID tenant wizard](entra_id_tenant_add_summary.md).

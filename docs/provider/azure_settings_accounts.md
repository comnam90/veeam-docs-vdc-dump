# Managing Veeam Service Account


When you connect Veeam Data Cloud to your Azure tenant account, Veeam Data Cloud creates an Azure application that acts as a service principal in your tenant. Veeam Data Cloud uses this service account to protect your resources.

In Veeam Data Cloud for Microsoft Azure, you can view information about the Veeam service account, as well as add new subscriptions in your Azure tenant to the protection scope of Veeam Data Cloud or update permissions.

# Creating EC2 Backup Policies


To create a backup policy, complete the following steps:

1. [Launch the Add EC2 Policy wizard](aws_backup_create_ec2_launch.md).
2. [Specify a backup policy name and description](aws_backup_create_ec2_name.md).
3. [Configure backup source settings](aws_backup_create_ec2_source_settings.md).
4. [Enable guest processing](aws_backup_create_ec2_guest_processing.md).
5. [Configure protection settings](aws_backup_create_ec2_source_settings.md).
6. [Review default policy settings](aws_backup_create_ec2_default_settings.md).
7. [Enable AWS tags assigning](aws_backup_create_ec2_tags.md).
8. [Specify notification settings](aws_backup_create_ec2_notifications.md).
9. [Finish working with the wizard](aws_backup_create_ec2_finish.md).

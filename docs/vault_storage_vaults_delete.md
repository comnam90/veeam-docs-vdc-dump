# Deleting Storage Vaults


Veeam Data Cloud Vault users can permanently delete storage vaults from the backup infrastructure. This operation depends on whether the storage vault is empty or contains data. The following options are available:

* [Deleting an empty storage vault](vault_storage_vaults_delete_empty.md)

This operation is available in the Veeam Data Cloud customer portal.

* [Deleting a non-empty storage vaults](vault_storage_vaults_delete_nonempty.md)

This operation requires assistance from Veeam Customer Support.

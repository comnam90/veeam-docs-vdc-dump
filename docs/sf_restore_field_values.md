# Restoring Field Values

In this article

Field value restore allows you to recover earlier versions of changed or deleted field values. Using this type of restore, you can restore values of existing fields only. If a field no longer exists in Salesforce, restore Salesforce metadata first. For details, see [Restoring Metadata](sf_restore_metadata.md).

To restore field values, take the following steps:

1. [Launch the Restore Field Values wizard](sf_restore_field_values_launch.md).
2. [Specify a name and description for the restore job](sf_restore_field_values_name.md).
3. [Select a target Salesforce tenant](sf_restore_field_values_organization.md).
4. [Select records whose field values you want to restore](sf_restore_field_values_data.md).
5. [Select fields whose values will be restored](sf_restore_field_values_fields.md).
6. [Configure additional restore settings](sf_restore_field_values_options.md).
7. [Check permissions](sf_restore_field_values_verification.md).
8. [Finish working with the wizard](sf_restore_field_values_summary.md).

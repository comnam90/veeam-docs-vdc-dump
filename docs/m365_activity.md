# Activity


Veeam Data Cloud for Microsoft 365 offers a full overview of activities for each Veeam Data Cloud for Microsoft 365 tenant. In the Activity page of a tenant, you can view Veeam Data Cloud and user actions, backup and restore session information and audit log events. For each action, you can also view a detailed action log.

Only users with the OrganizationAdmin or M365:Administrator roles or a custom role with the View Activity Logs permission can view the Activity page of each tenant. For more information about roles, see [Roles](users_roles.md).

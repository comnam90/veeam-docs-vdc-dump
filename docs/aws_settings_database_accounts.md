# Managing Database Accounts


To allow Veeam Data Cloud for AWS to authenticate against PostgreSQL DB instances protected by RDS backup policies, you must specify credentials of database accounts that will be used to access databases of the DB instances when performing [image-level backup](aws_backup_create_rds_processing_settings.md) and [RDS database restore](aws_restore.md).

# Adding Database Accounts


To add a new database account, do the following:

1. [Launch the Add Database Account wizard](aws_settings_database_account_wizard.md).
2. [Specify an account name and description](aws_settings_database_account_name.md).
3. [Specify general settings](aws_settings_database_account_password.md).
4. [Finish working with the wizard](aws_settings_database_account_summary.md).

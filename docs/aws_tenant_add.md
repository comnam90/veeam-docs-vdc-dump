# Creating Tenants


To create a tenant in Veeam Data Cloud for AWS, do the following:

1. [Launch the Add AWS Tenant wizard](aws_tenant_add_launch.md).
2. [Configure access to AWS services and resources using a CloudFormation template](aws_tenant_connection.md).
3. [Specify a name for the tenant and choose AWS accounts.](aws_tenant_scope.md)
4. [Choose a geographic location](aws_tenant_region.md).
5. [Finish working with the wizard](aws_tenant_finish.md).

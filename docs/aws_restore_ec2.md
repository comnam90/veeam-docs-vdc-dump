# EC2 Restore


Veeam Data Cloud for AWS offers the following restore options:

* [Instance restore](aws_restore_ec2_entire.md) — restore an entire EC2 instance.
* [Volume restore](aws_restore_ec2_volume.md) — restore EBS volumes attached to an EC2 instance.

You can restore EC2 instance data to the most recent state or to any available restore point.

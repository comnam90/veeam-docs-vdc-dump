# Azure Resource Providers


Veeam Data Cloud for Microsoft Azure requires Azure subscriptions to be registered for the following resource providers:

* Microsoft.Authorization
* Microsoft.Commerce
* Microsoft.Compute
* Microsoft.KeyVault
* Microsoft.Network
* Microsoft.Resources
* Microsoft.Storage
* Microsoft.Sql

For more information on registering an Azure subscription for a resource provider, see [Microsoft Docs](https://learn.microsoft.com/en-us/azure/azure-resource-manager/management/resource-providers-and-types#register-resource-provider).

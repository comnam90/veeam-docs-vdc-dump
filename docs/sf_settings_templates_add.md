# Adding Data Masking Templates


To add a new data masking template, use the Add Data Masking Template wizard:

1. [Launch the Add Data Masking Template wizard](sf_settings_templates_add_launch.md).
2. [Specify the name of the new template](sf_settings_templates_add_name.md).
3. [Verify the Salesforce tenant](sf_settings_templates_add_organization.md).
4. [Specify the data that you want to mask](sf_settings_templates_add_data.md).
5. [Finish working with the wizard](sf_settings_templates_add_summary.md).

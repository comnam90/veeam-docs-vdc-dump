# Backup Storage Regions


This section lists the available regions for Veeam Data Cloud for Microsoft 365 Flex backup data storage.

|  |
| --- |
| note |
| Veeam Data Cloud for Microsoft 365 Express backup data is stored in the Microsoft-specific region based on your Microsoft 365 configuration and subscription. This region will be automatically represented by the Veeam Data Cloud region of your Veeam Data Cloud for Microsoft 365 configuration. For more information about Microsoft regions, see [this Microsoft article](https://learn.microsoft.com/en-us/azure/reliability/regions-list?tabs=all). |

# Editing Backup Policies


You can edit a backup policy that was automatically created when adding the Salesforce tenant.

To edit a backup policy, use the Edit Backup Policy wizard:

1. [Launch the Edit Backup Policy wizard](sf_backup_policies_edit_launch.md).
2. [Configure Salesforce API limits and verify user permisions](sf_backup_policies_edit_connection.md).
3. [Configure policy schedule and backup options](sf_backup_policies_edit_backup.md).
4. [Edit encryption settings](sf_backup_policies_edit_encryption.md).
5. [Edit retention settings](sf_backup_policies_edit_retention.md).
6. [Finish working with the wizard](sf_backup_policies_edit_summary.md).

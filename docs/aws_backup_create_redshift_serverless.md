# Creating Redshift Serverless Backup Policies


To create a backup policy, complete the following steps:

1. [Launch the Add Redshift Serverless Policy wizard](aws_backup_create_redshift_serverless_launch.md).
2. [Specify a backup policy name and description](aws_backup_create_redshift_serverless_name.md).
3. [Configure backup source settings](aws_backup_create_redshift_serverless_source_settings.md).
4. [Configure protection settings](aws_backup_create_redshift_serverless_protection_settings.md).
5. [Review default policy settings](aws_backup_create_redshift_serverless_default_settings.md).
6. [Enable AWS tags assigning](aws_backup_create_redshift_serverless_tags.md).
7. [Specify notification settings](aws_backup_create_redshift_serverless_notifications.md).
8. [Finish working with the wizard](aws_backup_create_redshift_serverless_finish.md).

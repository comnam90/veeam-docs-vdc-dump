# Microsoft Entra ID Protection


Veeam Data Cloud for Microsoft Entra ID is a backup as a service solution developed to deliver data protection and disaster recovery for Microsoft Entra ID. With Veeam Data Cloud for Microsoft Entra ID, you can perform the following operations:

* Create backups of Microsoft Entra ID tenants.
* Create backups of Microsoft Entra ID audit and sign-in logs.
* Restore users, groups, administrative units, roles, applications, service principals, Conditional Access policies and Microsoft Intune policies from Microsoft Entra ID tenant backups to the Microsoft Entra ID environment.
* Restore properties of users, groups, administrative units, roles, applications, service principals, Conditional Access policies and Microsoft Intune policies from Microsoft Entra ID tenant backups to the Microsoft Entra ID environment.
* Export audit and sign-in logs from Microsoft Entra ID log backups.

* Export users, groups, administrative units, roles, applications, service principals, Conditional Access policies, Microsoft Intune policies and organization contacts from Microsoft Entra ID tenant backups to files in JSON format.

# EFS Restore


Veeam Data Cloud for AWS offers the following restore options:

* [File system restore](aws_restore_entire_efs.md) — restore an entire Amazon EFS file system.
* [File-level recovery](aws_restore_efs_item.md) — recover individual files and folders stored in a file system.

You can restore EFS file system data to the most recent state or to any available restore point.

# Step 6. Configure File System Settings


The list of settings that you can configure for a restored file system depend both on the file system type and the option you choose at the Choose Restore Mode step of the wizard.

|  |
| --- |
| Important |
| Veeam Data Cloud for AWS does not allow you to change the type of Microsoft Active Directory to which the restored file system will be joined. |

# Adding Microsoft Azure Tenants


To start protecting your Microsoft Azure resources, you must add your Microsoft Azure tenant to Veeam Data Cloud.

|  |
| --- |
| Note |
| If you are a customer of a Veeam Cloud & Service Provider partner, the Add Azure Tenant wizard is not available for direct use. Ask your service provider to add your Azure tenant or send you an email with an invitation link that allows you to log in to Veeam Data Cloud and launch the wizard. |

To add an Azure tenant, you must perform the following steps:

1. [Launch Add Tenant wizard](azure_tenant_add_launch.md).
2. [Connect Veeam Data Cloud to Azure Tenant](azure_tenant_add_connect.md).
3. [Select Azure subscription](azure_tenant_add_subscription.md).
4. [Select backup management region](azure_tenant_add_region.md).
5. [Review tenant settings](azure_tenant_add_summary.md).
